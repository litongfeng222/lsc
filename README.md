# LSCgroup 📚

> Learning · Share · Communicate

**北京师范大学附属中学 · 初三学习小组资料站**

一个轻量、美观、可扩展的资料共享网站，专为学习小组设计。

---

## ✨ 功能

- ✅ **资料浏览** — 按科目分类，一目了然
- ✅ **文件下载** — 点击即可下载资料
- ✅ **手机适配** — 无论手机还是电脑都能用
- ✅ **快速更新** — 发资料给姐姐，网站自动更新
- 🧩 **可扩展** — 预留了打卡、留言、搜索等模块接口

## 🚀 部署指南（一键部署到 Vercel）

### 前提
1. 注册 [GitHub](https://github.com) 账号（免费，13岁以上可注册）
2. 注册 [Vercel](https://vercel.com) 账号（用GitHub登录即可）

### 步骤
```bash
# 1. 创建 GitHub 仓库
# 在 GitHub 上新建仓库，命名为 LSCgroup

# 2. 上传代码
git clone https://github.com/你的用户名/LSCgroup.git
# 把本项目的所有文件复制进去
git add .
git commit -m "🎉 初版上线"
git push

# 3. 部署到 Vercel
# 登录 vercel.com → 导入 LSCgroup 仓库 → 部署
# 等1分钟，搞定！
```

部署完成后，你会得到一个 `https://lscgroup.vercel.app` 类似的链接，分享给小组同学即可！

## 📦 如何更新资料

把文件通过微信发给**姐姐**，姐姐帮你：
1. 筛选有用资料
2. 上传到仓库
3. 网站自动更新

同学们打开链接就能看到新资料，无需任何操作。
